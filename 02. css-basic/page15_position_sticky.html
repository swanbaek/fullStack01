sticky 네비게시션
position: sticky는 요소의 스크롤 위치에 따라 동적으로 동작하는 특수한 position 값입니다. sticky는 relative와 fixed의 특성을 조합한 것 
            1. 기본 동작
                스크롤 범위 내에서는 relative처럼 동작:
                요소는 원래의 문서 흐름에 따라 배치됩니다.
                스크롤 범위를 벗어나면 fixed처럼 동작:
                요소가 설정된 특정 임계점에 도달하면 고정된 위치에 머무르게 됩니다.
            2. 조건 및 동작 방식
                position: sticky를 사용하려면 **상위 컨테이너(스크롤 부모)**와의 관계를 설정해야 합니다.

            주요 속성:
                top, right, bottom, left 중 하나 이상을 설정해야 합니다.
                이 속성들은 스크롤 시 요소가 고정될 위치를 결정합니다.
                동작 과정:
                기본 상태 (relative):
                요소는 상위 컨테이너 안에서 일반적으로 배치됩니다.
                임계점 도달 (fixed):
                스크롤이 진행되며 설정한 top, right 등의 값에 도달하면, 해당 위치에 고정됩니다.
                범위 종료:
                요소의 상위 컨테이너 경계를 벗어나면 더 이상 고정되지 않고 자연스럽게 스크롤됩니다.
